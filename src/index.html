About TechTrends

TechTrends is a mashup web app that reveals where in the world programming languages are trending, based on GitHub activity and Hacker News discussions. It's built with real-time data and aims to inspire developers to explore what's hot, where, and why.

A Mashup

A mashup is a web app that blends data from multiple sources into a new, interactive experience. It feels like one unified tool, even though the data comes from elsewhere.

This mashup uses:

  • GitHub REST API
  • Hacker News API

GitHub is a platform where developers share and collaborate on code.
Hacker News is a tech-focused news site where people share and discuss trends in programming and startups.

About the Programmer

I'm Sara Sjodin Scolari, a developer with a passion for exploring how data can be visualized and combined in meaningful ways. This mashup was built to experiment with secure API handling, design, and responsive layouts using modern web tools like Parcel and SCSS.
